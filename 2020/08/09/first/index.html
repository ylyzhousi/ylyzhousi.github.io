Tomcat 9.0.24下载与安装最新版教程

百度云地址：https://pan.baidu.com/s/1sxdBa6GA281ZArOUwdZJCw，提取码：77dj

直接下载应用程序安装：

Windows x64安装，

1、安装方法/步骤：

（1）双击运行安装包，打开安装向导界面，点击Next；

（2）这是许可协议界面，一般不需要阅读，直接点击（你懂得）I Agree按钮进入下一个界面；

（3）这一步我建议不需要操作，默认即可，点击Next进入下一个界面；

（4）这里是配置信息，包括端口号和账号密码。建议设置管理员账号密码，密码设置好之后点击Next进入下一步；

（5）这里是java的JDK的安装目录路径，如果没有自动获取的话需要你点击…按钮进行手动选择，然后点击Next进入下一步；

（6）接下来选择安装的目录路径文件夹，建议直接默认路径就可以，避免出错，然后点击Install进行安装；

（7）等待安装完成后，点击Finish按钮，关闭对话框，自此软件安装完成。

2、环境变量的配置步骤

通常情况下，是先下载JDK、JRE后再下载Tomcat，所以在安装配置Tomcat之前，需要先检查JDK、JRE的环境变量是否配置完成。具体略。

如果上述没问题，正式进入Tomcat的环境变量配置，步骤如下：

（1） 计算机—属性—高级系统设置—高级—环境变量—系统变量—新建

​ 变量名：CATALINA_HOME

​ 变量值：C:\Program Files\Apache Software Foundation\Tomcat 9.0

​ (即为Tomcat9.0的默认安装路径)

!
（2）计算机—属性—高级系统设置—高级—环境变量—系统变量—新建
变量名：CATALINA_BASE
变量值：C:\Program Files\Apache Software Foundation\Tomcat 9.0
(即为Tomcat9.0的默认安装路径)
!

（3） 配置path

​ 变量值：# ;%CATALINA_HOME%\bin;%CATALINA_HOME%\lib #

​ （#里边的内容都需要，在path最后边加）

​

!

至此，环境变量配置完毕。

3、检测Tomcat是否安装配置成功

（1） 使用快捷键打开cmd命令；

!

（2）在命令行输入startup.bat，按回车键，可以看到一下信息，同时会弹出另一个dos命令窗口，该窗口一闪而过，说明启动成功；

!
